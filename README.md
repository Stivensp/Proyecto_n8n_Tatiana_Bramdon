# Proyecto_n8n_Tatiana_Bramdon
Asistente inteligente que consulta el clima de una ciudad y genera recomendaciones de vestimenta según las condiciones meteorológicas actuales. Utiliza n8n para la automatización de flujos, una API meteorológica para obtener datos en tiempo real, MySQL para almacenar consultas y MCP para estructurar la interacción entre servicios.
